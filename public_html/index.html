What is the DC Code?

The Code of the District of Columbia is the compilation of District’s statutory laws, which derive from acts of the Council of the District of Columbia and federal statutes, including the Home Rule Act.

The DC Code does not include laws that are temporary in nature (including temporary or emergency legislation) or regulations (which are found in the DCMR).

Please head over to official.dccode.gov/current to browse the latest publication of the DC Code.

The DC Code is made available in an open electronic data format to encourage software developers and legal publishers to build other modern tools for promoting public and equitable access to the District’s laws.

Electronic Code Update Chart

The electronic code update chart is a machine-readable set of instructions for how the DC Code has been updated by each codified act. Access the electronic code update chart at https://github.com/....

The update chart repository also acts as a public audit log for all changes to the DC Code. Each change to the DC Code is recorded in the code update chart in perpetuity.

Public Domain Dedication
